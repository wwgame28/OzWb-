# OzWb Rare Discount Bot

Telegram-бот для мониторинга категорий Ozon и Wildberries и поиска товаров с аномально низкой ценой относительно собственной истории цен.

## Что умеет MVP

- хранит историю цен товаров;
- ежедневно запускает сканирование категорий;
- считает базовую цену по медиане или среднему за последние 30 дней;
- определяет редкую скидку по реальному снижению цены, а не по зачеркнутой цене маркетплейса;
- отправляет карточки находок в Telegram;
- содержит расширяемые коллекторы для Ozon/Wildberries.

> В репозитории добавлен безопасный MVP: по умолчанию включен `DemoCollector`, чтобы бот можно было запустить без риска блокировок. Реальные коллекторы Ozon/WB вынесены в отдельные классы-заготовки, которые нужно адаптировать под актуальные публичные XHR/API-эндпоинты и условия использования площадок.

## Быстрый старт

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
cp .env.example .env
```

Заполните `TELEGRAM_BOT_TOKEN` в `.env`, затем:

```bash
python -m app.bot
```

Для Playwright, если будете использовать браузерный парсинг:

```bash
playwright install chromium
```

## Команды бота

- `/start` — регистрация пользователя и краткая справка;
- `/categories` — показать категории мониторинга;
- `/scan` — запустить ручное сканирование и получить найденные скидки.

## Логика редкой скидки

Если товар стоил около 10 000 ₽ последние 30 дней, а сейчас стоит 7 900 ₽, бот считает реальную скидку примерно 21%.

```python
from app.discount import detect_rare_discount

result = detect_rare_discount(
    current_price=7900,
    historical_prices=[9990, 10100, 10050, 9800, 10200, 9950, 10000, 10150, 9900, 10020],
    threshold=0.20,
)
print(result)
```
